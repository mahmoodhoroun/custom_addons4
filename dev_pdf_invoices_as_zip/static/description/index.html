Download Invoices Reports as Zip File

This Odoo application will allows you to download pdf reports of Invoices/Bills as zip file at desired location in your system

More Apps Ask for Demo

FEATURES

  • Download pdf reports of Invoices/Bills as zip file at desired location in your system
  • Configure Directory(folder path) in which pdf reports invoices/Bills will be downloaded
  • Specify Split By which will decides how many invoices/Bills will be in one zip file
  • Such as if you are going to download pdf reports of 4 invoices/Bills and specified 2 in Split By then 2 zip files will be created at configured location, and each zip file will contain 2 pdf reports of invoices/bills
  • There is also an option which allows you to download multiple reports of invoices/bill into one pdf files, means 2 zip files will be created at configured location, and each zip file will contain 1 pdf file and in that pdf file there will be 2 reports of invoices/bills
  • There is another option which allows you to download files at desired location as well as it allows you to download files from the screen also
  • Added Filter in invoices/bills screen:
    • Report Downloaded : Shows those invoices/bills whose pdf report is downloaded as zip file
    • Report not Downloaded : Shows those invoices/bills whose pdf report is not downloaded as zip file
  • Note : Only user with Allow to Download Invoices/Bills Report(pdf) as Zip right can configure Directory(folder path), Split By and download pdf reports of invoices/bills
  • Note : You can only download reports of those invoices/bills which are either Open or Paid
