1. To access the OCR service, please follow the steps below:

  • Create an Account & Retrieve Your Key: Visit https://ai.fynix.app register, and access your Authentication Key from the OCR Dashboard.
  • Activate Your OCR Service: Purchase the required OCR tokens from https://ai.fynix.app/shop to start using the service.

2. Setup & Configuration

Install Odoo OCR Using AI - Invoices & Bills Module
From App Menu > Install Module

3. API Configuration Inside Odoo

After activating Developer mode, go to Settings => OCR AI Integration as mentioned in the below screenshot.

4. Configure Odoo OCR AI Token

You can configure company based OCR AI Authentication Token which you have received from Service Portal.
Server URL MUST BE : https://ai.fynix.app/tus_ocr_api

5. Dynamic Mapping of Fields of Invoice Model

In order to map your dynamic fields with Invoice's data go to Settings => OCR AI Integration => Model Configuration => In Model Select Journal Entry for Invoice and Vendor Bill configuration as per the below screenshot.
